Speedrunning

What is Speedrunning?

Speedrunning it the art of beating a game as fast as possile, achieving a world record. The most popular format of speedrunning a game is call any%, in which the player is tasked with reaching the end of the game as fast as possible, skipping any parts that aren't required to beat the game. On the contrary, the second most popular speedrunning category, 100%, requires the player to collect everying in the game before beating the game, weither this be beating every level in the game as is the case for majority of platofmer games or unlocking and beating the game with all playable characters as is the case with most rogue-like games.

How do I start Speedrunning?

The best place to start is to first think about what video games you like playing and find one you would think would be fun to beat as fast as possible. Remember that you should think about the length of the game as well and if you want to spend hours trying to beat the game or just a couple minutes. Both have their advantages: Longer games are easier to improve on as they allow for more mistakes, whilst short games allow for more runs to be completed over time, which allows for more focused areas for improvement and opportunities to get better times. Next go to speedrun.com and find the game you chose to speedrun. There are thousands of communites representing thousands of different games, so find the community belonging to the game you chose and search for tutorials on how to speedrun the game. Finally it's all down to dedication and practise. Start of small by learning individual parts of the speedrun, then add all parts together and being a full run.

LiveSplit

LiveSplit is a useful tool for speedrunning which times your runs of the game you are playing and enables you to split the game you are playing into segments and time each individual segment. Not only can you manually time your progress in a game but "LiveSplit will automatically detect if Game Time and/or Auto Splitting is available for a game and let you activate it in the Splits Editor. Game Time is automatically read directly from an emulator or PC game, and you can use it by switching to Game Time under Compare Against." (Sourced from the offical LiveSplit website). Livesplit can also keep track of all your runs, including your personal best runs which can be compared to in future runs. I would highly recommend downloading LiveSplit for speedrunning and recommend watching a Youtube tutorial on how to set up LiveSplit.
